# Why Multi-Agent Here?

Single evaluator agents fail because:

* evaluation dimensions conflict
* safety != factuality
* latency != quality
* retrieval != reasoning
* one judge becomes biased

So specialized evaluator agents independently score dimensions.

This creates:

* ensemble evaluation
* critic-based consensus
* weighted governance

```json
{
  "correctness": {
    "0": "Incorrect answer",
    "1": "Partially correct",
    "2": "Mostly correct",
    "3": "Fully correct"
  },

    "0": "No grounding",
```

* judge agents become inconsistent

* metrics become meaningless
